Create `src/ui/CardRenderer.js` with a single exported function:

```javascript
export function createCardElement(card)
```

The function receives a card object `{ suit, rank, faceDown }` and returns a DOM element with this structure:

```html
<div class="bj-card [bj-card--flipped if faceDown]">
  <div class="bj-card-inner">
    <div class="bj-card-front bj-card--red|bj-card--black">
      <span class="bj-card-corner bj-card-corner--top">RANK<br>SUIT_SYMBOL</span>
      <span class="bj-card-suit">SUIT_SYMBOL</span>
      <span class="bj-card-corner bj-card-corner--bottom">RANK<br>SUIT_SYMBOL</span>
    </div>
    <div class="bj-card-back"></div>
  </div>
</div>
```

Suit symbol mapping:
- hearts -> '\u2665' (black heart, displayed in red via CSS class)
- diamonds -> '\u2666'
- clubs -> '\u2663'
- spades -> '\u2660'

Color class logic:
- hearts or diamonds -> add class `bj-card--red` to `.bj-card-front`
- clubs or spades -> add class `bj-card--black` to `.bj-card-front`

If `card.faceDown` is true, add class `bj-card--flipped` to the outer `.bj-card` element.

Also export a helper:
```javascript
export function formatChips(cents)
```
Returns `$X` string where X is cents/100 formatted with toLocaleString('en-US', { minimumFractionDigits: 0 }). Example: formatChips(85000) returns "$850", formatChips(100000) returns "$1,000".
